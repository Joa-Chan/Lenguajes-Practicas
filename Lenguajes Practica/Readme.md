## Qué hace el programa

Son dos versiones del juego piedra, papel o tijera contra la computadora.

La primera versión es básica, jugas 3 rondas fijas contra la PC.

La segunda versión tiene mejoras:
- Se puede configurar para jugar mejor de 5 (o las que quieras cambiando la variable)
- El juego termina antes si ya ganaste o perdiste sin posibilidad de empate
- Si escribes mal la jugada te vuelve a preguntar sin contar la ronda

## Cómo usar

Necesitas tener Python instalado.

Para jugar la versión 1:
```
python piedra_papel_tijera.py
```

Para jugar la versión 2:  
```
python piedra_papel_tijerav2.py
```

Cuando te pregunte tu jugada escribí: piedra, papel o tijera
